# TrollFools Repository

这是一个iOS越狱插件仓库，提供各种系统定制和功能增强插件。

## 仓库信息

- **仓库名称**: 宝藏巨魔仓库
- **维护者**:  armyb
- **支持系统**: iOS 14.0-17.0

## 添加仓库

在TrollFools SLPEdition设备上添加此仓库：

```
https://armyb.github.io/TrollFools/
```

## 可用插件示例

### TGMask
- **版本**: v1.0.0
- **描述**: 一个伪装插件，输入当前时间进入解锁进入Telegram。
- **大小**: 1.5MB
- **作者**: iosrxwy

### BZhelper
- **版本**: v1.6.7
- **描述**: 宝藏广告助手，广告加速、广告屏蔽，等等。
- **大小**: 1.8MB
- **作者**: iosrxwy

### heiheipay
- **版本**: v1.0.0
- **描述**: 本地验证通用解锁插件，点击购买取消即可。
- **大小**: 1.2MB
- **作者**: iosrxwy
