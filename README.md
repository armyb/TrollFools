# TrollFools Repository

这是一个iOS越狱插件仓库，这是一个专门为 iOS 巨魔应用 TrollFools 收集、分享和存储自用破解或去广告插件的资源库。提供各种系统定制和功能增强插件。

## 仓库信息

- **仓库名称**: 巨魔插件仓库
- **维护者**:  armyb
- **支持系统**: iOS 14.0-17.0

## 添加仓库

在TrollFools SLPEdition设备上添加此仓库：

```
https://armyb.github.io/TrollFools/
```

## 文件结构

```
├── trollpackages.json          # 主API文件
├── icons/                      # 图标文件夹
├── downloads/                  # 下载文件夹
├── screenshots/                # 截图文件夹
└── repo-icon.png              # 仓库图标
```
